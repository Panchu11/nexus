## 🚀 **Deploy in 5 Minutes**

### Quick Steps:

1. **Create Vercel Account**
   - Go to https://vercel.com
   - Sign up with GitHub

2. **Import Repository**
   - Go to https://vercel.com/dashboard
   - Click **Add New** → **Project**
   - Select `nexus` repository
   - Set Root Directory to `sentient-team`
   - Click **Deploy**

3. **Add Environment Variables**
   - Go to **Settings** → **Environment Variables**
   - Add your Supabase credentials (see below)
   - Click **Save**

4. **Redeploy**
   - Go to **Deployments**
   - Click three dots on latest deployment
   - Click **Redeploy**

5. **Test**
   - Click deployment URL
   - Test all features
   - Done! 🎉

## 📞 **Need Help?**

### Common Issues:

**Build failed?**
- Make sure Root Directory is set to `sentient-team`

**Database not connecting?**
- Check environment variables are correct
- Verify Supabase project exists

**Theme toggle not working?**
- Clear browser cache
- Refresh the page

**Comments not saving?**
- Verify comments table exists in Supabase
- Check database permissions
